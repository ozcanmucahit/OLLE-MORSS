## Yüklemek için

Node sürümünüz yoksa [Node.js](https://nodejs.org/) Kurun
Install the dependencies and devDependencies and start the server.

```sh
path: cd OLLE-MORS
install: npm i or npm install -g vite
start: npm run dev
```
